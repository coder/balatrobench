Submit Your Strategy

Share your custom strategy with the community by submitting a pull request to our GitHub repository.

Quick Submission Guide

1. Fork & Clone

Fork the repository and clone it to your local machine

2. Create JSON

Add your strategy as a JSON file in the data/strategies/ folder

3. Submit PR

Create a pull request with your strategy data

1 Fork the Repository

Start by forking the BalatroBench repository on GitHub:

git clone https://github.com/YOUR_USERNAME/balatrobench-site.git
cd balatrobench-site

2 Create Your Strategy File

Create a new JSON file in the data/strategies/ folder:

// data/strategies/your-strategy-name.json
{
  "title": "Your Strategy Name",
  "author": "YourUsername",
  "model": "GPT-4o / Claude-3.5-Sonnet / etc",
  "score": "8.5",
  "winRate": "75%",
  "avgTokens": "15000",
  "date": "2024-01-20",
  "description": "Brief description of your strategy",
  "prompt": "Your full system prompt here...",
  "methodology": "Explanation of your approach",
  "results": {
    "seeds": [1, 2, 3, ...],
    "scores": [8.2, 8.8, 8.1, ...]
  }
}

3 Submit Pull Request

Commit your changes and create a pull request:

git add data/strategies/your-strategy-name.json
git commit -m "Add [Your Strategy Name] community submission"
git push origin main

Then create a pull request on GitHub with the title: "Community Submission: [Your Strategy Name]"

Submission Requirements

  • Must include valid benchmark results on standard seeds
  • Strategy description must be clear and detailed
  • JSON file must follow the required schema
  • Results must be reproducible
  • No offensive or inappropriate content
