Leaderboard Columns Explained

Ranking & Model Info

Rank: Sorted by average rounds reached (highest first)
Model: OpenRouter model names with creator-suggested parameters for open-source models, defaults for closed-source
Provider: Model developer/organization

Performance Metrics

Rounds: Average rounds reached across multiple games
Completed: Success rate for round completion (rounds stopped at 3 consecutive errors/failed calls)

Call Quality Breakdown

Success: Valid tool calls that execute successfully in game state
Error: Invalid responses (no tool call, JSON errors, plain text responses)
Failed: Valid tool calls that can't execute (e.g., discarding 6 cards when limit is 5)

Efficiency Metrics

Input/Output Tokens: Token counts per tool call (including reasoning and tool call tokens)
Time per Call: Average LLM response generation time
Cost per 1K Calls: Pricing based on cheapest OpenRouter option (scaled to 1K for visual comparison)
